Многоуровневая VR арена

Цели проекта

  • Проработка пользовательских сценариев взаимодействия с ключевыми (конверсионными) интерфейсами сайта
  • Проработка UX-проекта с учетом пользовательских сценариев
  • Разработка концепции дизайна в соответствии с корпоративным стилем компании «РУСАЛ»
  • Создание удобного каталога с возможностью настройки фильтров в соответствии со спецификой товаров
  • Проработка нескольких сценариев поиска, в том числе подбора товаров по модели автомобиля
  • Разработка личного кабинета для физических и юридических лиц

Задачи

  • Проработка пользовательских сценариев взаимодействия с ключевыми (конверсионными) интерфейсами сайта
  • Проработка UX-проекта с учетом пользовательских сценариев
  • Разработка концепции дизайна в соответствии с корпоративным стилем компании «РУСАЛ»
  • Создание удобного каталога с возможностью настройки фильтров в соответствии со спецификой товаров
  • Проработка нескольких сценариев поиска, в том числе подбора товаров по модели автомобиля
  • Разработка личного кабинета для физических и юридических лиц

Решения

Рынок электронной коммерции в сфере B2B постепенно развивает цифровые каналы коммуникации. За 1,5 года появились новые сервисы с продуманным для бизнеса функционалом. Однако пока уровень цифровизации этого сектора рынка гораздо ниже, чем в розничной торговле. Это обусловлено нюансами сферы. Одна из главных проблем для производственных гигантов — перенести логику работы в цифровую плоскость. Для корректного отображения данных и базового функционала B2B-порталов требуется интеграционный слой. А для этого необходимо отладить логику внутренних процессов и подготовить системы к интеграции. В металлургической отрасли, где продается и поставляется товар со специфическими характеристиками, бывает непросто организовать тот порядок, который нужен для цифровизации. Сложность повышается в случае, когда для каталога маркетплейса необходимо связать учетные системы разных компаний.
